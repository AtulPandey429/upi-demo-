# upi-demo-

UPI payment demo with PhonePe (React + Express).

## Setup

```bash
npm run install:all
cp .env.example .env
# Edit .env with PhonePe keys
npm run dev          # API on :5000
npm run dev:frontend # UI on :5173
```
